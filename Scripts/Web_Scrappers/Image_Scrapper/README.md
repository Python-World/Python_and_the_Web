# Images Scraper
#### This script scrapes images from a URL and stores them on your local folder.

# Pre-requisites:
#### Run the following command:
```python
pip install -r requirements.txt
```
# Instructions to run the script:
#### Run the command:
```python
python Scraper.py
```
